Scientific Paper
================

This repository contains the data and R code used in the statistical analysis paper of the paper entitled "Contribution of chromosomes 1H^chS and 6H^chS to fertility restoration in the wheat msH1 CMS system under different environmental conditions"

The fertility data are found: 
"fpiston/Paper_CMStmp/code_data/fertility_environment.csv"

The R code used in the fora nalysis and plots is:
"fpiston/Paper_CMStmp/code_data/cmstemp_paper-code.R"
